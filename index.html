Введение

Актуальность темы: В современном мире веб-технологии являются фундаментом цифровой трансформации. Умение создавать качественные, интерактивные и отзывчивые веб-интерфейсы — ключевой навык фронтенд-разработчика. Данная работа демонстрирует практическое освоение этих навыков.

Цель работы: Систематизировать и продемонстрировать полученные знания и умения в области HTML, CSS и JavaScript через серию завершенных учебных и практических проектов.

Задачи:

  • Изучить основы семантической верстки на HTML5.
  • Освоить современные методы стилизации с помощью CSS3, включая Flexbox и Grid.
  • Реализовать интерактивные элементы с использованием vanilla JavaScript.
  • Создать адаптивные веб-интерфейсы, корректно отображающиеся на различных устройствах.
  • Применить полученные знания для решения практических задач.

Методы и технологии: В процессе работы использовались HTML5, CSS3, JavaScript, а также инструменты контроля версий Git. Для создания адаптивного дизайна применялся подход Mobile First.
